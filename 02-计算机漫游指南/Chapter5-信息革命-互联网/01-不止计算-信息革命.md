# 新的革命

通常所说的信息革命值得可不是计算机，而是互联网。（当然互联网的组成通常都是计算机。）

计算机掀起了算力的革命。



# OSI模型

> Open System Interconnection Reference Model
>
> 由国际标准化组织提出，一个试图使计算机在世界范围内连伟网络的标准框架。

OSI模型的重要性相比于冯诺依曼结构不遑多让。



> 没有什么复杂的模型是不能通过添加一个中间层来简化或解耦的，如果有，那就加两个。——尼古拉·吴

可见网络通信协议模型确实复杂，足足分了七层，才能够让各种应用、服务有条不紊地运行。

而基于这个七层模型的现代计算机网络也确实不负所望地创造了一个又一个奇迹。
