# 汇编语言

```text
0100 0001 ；寄存器存入1
0001 0100 ；寄存器的数字加4
0010 0000 ；乘2
0001 0011 ；再加三
```

即`1000001000101000010000000010011`，啊这……显然让人看都不想看。

```text
MOV   1 ；寄存器存入1
ADD   4 ；寄存器的数字加4
SHL   0 ；乘2（介于我们设计的乘法器暂时只能乘2，这个0是占位的）
ADD   3 ；再加三
```

机器语言和汇编语言一一对应。至此也算是走到了汇编，下一节编译原理——从汇编到高级语言。
