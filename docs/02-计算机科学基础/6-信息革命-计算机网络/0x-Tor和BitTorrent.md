# Tor

The Onion Router，洋葱路由器





# .torrent种子文件

包含 Tracker 信息和文件信息两部分。

Tracker信息主要是BT下载中需要用到的Tracker服务器的地址和针对Tracker服务器的设置，文件信息是根据对目标文件的计算生成的，计算结果根据BitTorrent协议内的[Bencode](https://zh.wikipedia.org/wiki/Bencode)规则进行编码。
