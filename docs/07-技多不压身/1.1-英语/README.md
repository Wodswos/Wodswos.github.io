四项基本能力：听说读写。其中又以读为最核心、最基本的能力。

> 听和读都是**信息接收**的过程，说和写都是**信息表达**的过程。
>
> 任何语言的发展都是围绕信息的传递、或者信息的更好传递展开，无论是语法还是词汇，还是时态等细枝末节的东西。当然每门语言都会在具体的实践中有各种差异。
>
> 如在中文中可能更多地用“已经、曾经、正在”等词来直接表达时态，英文中则通过动词的变化来表达时态。
