# 奇怪的语言现象



“某人死之前”和“某人没死之前”，后者相较于前者加了一个否定，表达的却是同一个意思，而且大部分人都能自然地反应过来。

不只是汉语喔，英语也有类似“after dark”来表示日落之后。
