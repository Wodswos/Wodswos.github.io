# “设计”的艺术

整个计算机体系都是设计的艺术，任何一门子课程也都要在需求之间做到权衡。

数据结构（时间和空间）、算法（空间和时间）、操作系统（这要权衡的就多了）、软件工程（开发难度和代码效率），无一不需要考量权衡的艺术。
